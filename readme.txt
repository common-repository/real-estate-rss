== Description ==

Real Estate RSS is wordpress plugin to help you insert rss with love into your posts/pages.
No need coding or modify your themes to insert it.

<h3>Features</h3>
<ul>
<li>Custom Template.</li>
<li>Caching.</li>
<li>Multiple rss into posts/pages.</li>
</ul>


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `wordpress-tabs-slides` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Test and enjoy!
